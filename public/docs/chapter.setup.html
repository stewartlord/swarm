Swarm 2016.1 Guide (May 2016)

Setting up

This chapter covers the initial installation and configuration of Swarm. Also covered is upgrading Swarm if you have previously installed Swarm.

First, review the Runtime dependencies section. Note that Swarm's dependencies can mostly easily be installed using Swarm packages.

You have three choices for installing Swarm. Choose one approach:

  1. Use the Swarm packages (RPM or Debian):

    • Follow the steps provided in Swarm packages.

    • Once Swarm is installed, it needs some initial configuration (common to any installation approach). Skip to the Perforce configuration for Swarm section.

  2. Use the Swarm OVA (Open Virtualization Appliance):

    • Follow the steps provided in OVA configuration.

    • Once Swarm is installed, it needs some initial configuration (common to any installation approach). Skip to the Perforce configuration for Swarm section.

  3. Use the Swarm source distribution archive:

    • Follow all of the steps provided in the Initial manual installation and subsequent sections.

If you have not already done so, download Swarm: Download Swarm
